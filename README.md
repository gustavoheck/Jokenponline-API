<h1 align="center">🪨 📄 ✂️ Jokenpô Online API ✂️ 📄 🪨</h1>
 
Uma API REST desenvolvida para gerenciar partidas, jogadores e a lógica do clássico jogo Jokenpô (Pedra, Papel e Tesoura) em tempo real. 

> ⚠️ **Nota:** Este projeto está atualmente **em desenvolvimento**.
> 
## 🚀 Funcionalidades Essenciais Planejadas / Em Desenvolvimento

* Cadastro e autenticação de jogadores (JWT).
* Criação de salas de jogo (Matchmaking).
* Lógica de validação de jogadas (Quem ganha de quem).
* Histórico de partidas e placar geral (Leaderboard).
* Conexão via WebSockets para partidas em tempo real (Real-time).
